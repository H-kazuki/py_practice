#概要
_予定管理デスクトップアプリケーション_

#アプリケーションの機能一覧
1.予定の表示・追加・削除
2.カレンダーの表示
3.今日と明日の天気の表示

#環境
-言語
 -python3.8.5
-データベース
 -SQlite3
 
#ライブラリ
+requests(2.24.0)
+BeautifulSoup4(4.9.1)
+calendar　・・・標準ライブラリ
+datetime　・・・標準ライブラリ

#インストール
_ pythonをインストール後コマンドプロンプトに下記を貼り付ける_
+pip install beautifulsoup4
+pip install requests
